# Playwright JavaScript Project

This project uses [Playwright](https://playwright.dev/) for end-to-end browser testing with JavaScript.

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Run tests:
   ```sh
   npx playwright test
   ```

## Project Structure
- `tests/` - Example Playwright tests
- `playwright.config.js` - Playwright configuration

## Useful Commands
- `npx playwright test` - Run all tests
- `npx playwright test --headed` - Run tests with browser UI
- `npx playwright codegen` - Open Playwright code generator
